Cheap Managed Kubernetes with Terraform

18 April, 2019 at 14:15 -0600

Kubernetes is the way to deploy your services in a scalable and reliable way. However, it's a pretty complex system to manage yourself. Thankfully, cloud providers are offering managed versions where you only pay for the worker nodes. We'll use GKE, Google's managed kubernetes offering, to deploy a cluster so we can test out kubernetes. We'll use Terraform to make sure we have a repeatable deployment process.
